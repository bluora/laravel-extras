Provides extras for Laravel 5.

## Installation

Install via composer:

`composer require bluora/laravel-extras dev-master`

Add it to your config/app.php

```php
    'providers' => [
        ...
        Bluora\LaravelExtras\BladeDirectiveServiceProvider::class,
        ...
    ];
```
